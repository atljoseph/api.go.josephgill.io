
# api.josephgill.io

## Run

go run *.go

## Mock Database

## Commands

- `go run *.go` will run the `api` locally without docker.
- To use Docker locally, run `docker-compose up`, optionally provide the `--build` argument.
- `sh ./test.sh` will fire api calls at the server.
